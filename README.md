# Comparing Machine Learning Methods for Predicting Diseases from Symptoms in Healthcare

Bachelor's thesis (work in progress) — University of Europe for Applied Sciences, Germany.

This repository holds both the written thesis (LaTeX) and the experimental code that produced its results.

---

## Topic

A comparative study of five supervised classifiers for predicting diseases from patient-reported symptoms:

- Naive Bayes
- Logistic Regression
- Support Vector Machine
- Random Forest
- XGBoost

Models are trained and evaluated on two public Kaggle symptom-disease datasets, with hyperparameter tuning via `GridSearchCV` on the first dataset and fixed best-performing configurations applied to the second.
